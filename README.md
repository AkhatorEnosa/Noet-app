## Supabase Setup (One-time â€“ required for the app to work)

1. Create project â†’ https://supabase.com/dashboard
2. Go to **SQL Editor** and run:

   ```sql
      -- Create the notes table (your exact schema)
      CREATE TABLE public.notes (
        id          BIGINT GENERATED BY DEFAULT AS IDENTITY NOT NULL,
        created_at  TIMESTAMPTZ NOT NULL DEFAULT NOW(),
        data_value  VARCHAR NULL,                    -- main note content / body
        bg_color    TEXT NULL DEFAULT '',
        img         TEXT NULL DEFAULT '',
        index_num   BIGINT NULL,
        user_id     UUID NULL,                       -- links to auth.users
        pinned      BOOLEAN NULL DEFAULT FALSE,
        privacy     BOOLEAN NULL DEFAULT TRUE,
        title       TEXT NULL,
        CONSTRAINT  data_pkey PRIMARY KEY (id)
      ) TABLESPACE pg_default;
      
      -- Enable Row Level Security (very important for multi-user safety)
      ALTER TABLE public.notes ENABLE ROW LEVEL SECURITY;
      
      -- Policy: Users can only SELECT their own notes
      CREATE POLICY "Users can view their own notes"
        ON public.notes
        FOR SELECT
        USING (auth.uid() = user_id);
      
      -- Policy: Users can only INSERT notes for themselves
      CREATE POLICY "Users can insert their own notes"
        ON public.notes
        FOR INSERT
        WITH CHECK (auth.uid() = user_id);
      
      -- Policy: Users can only UPDATE their own notes
      CREATE POLICY "Users can update their own notes"
        ON public.notes
        FOR UPDATE
        USING (auth.uid() = user_id)
        WITH CHECK (auth.uid() = user_id);
      
      -- Policy: Users can only DELETE their own notes
      CREATE POLICY "Users can delete their own notes"
        ON public.notes
        FOR DELETE
        USING (auth.uid() = user_id);
   ```

3. (Optional) Enable Realtime on the `notes` table for future live sync.
